## 📂 폴더 구조

각 단원별 실습 소스 코드와 연습문제 풀이는 아래 폴더에 나누어 관리하고 있습니다.

* `01_search` : 기본 알고리즘 및 검색 (선형 검색, 이진 검색 등)
* `02_stack` : 스택(Stack) 구현 및 활용
* `03_queue` : 큐(Queue) 구현 및 링 버퍼 활용
* `04_recursive` : 재귀 알고리즘 (하노이의 탑, 8퀸 문제 등)
* `05_sort` : 정렬 알고리즘 (버블, 선택, 삽입, 셸, 퀵, 병합, 힙, 도수 정렬 등)
* `06_set` : 배열로 집합 구현하기

## 💻 개발 환경

* **Language** : C
* **Compiler** : GCC
* **WorkPlace** : [online gdb](https://www.onlinegdb.com/online_c_compiler)

  * 군 사지방 컴퓨터에서는 vs code의 다운로드가 불가함

## 📌 목표 및 규칙

1.  **가능한 실무에 가깝게** : 남이 봐도 이해가능하게 명시적이고 구조적으로 작성
