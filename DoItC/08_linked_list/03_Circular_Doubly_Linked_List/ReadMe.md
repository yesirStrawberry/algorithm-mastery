# 원형 이중 연결 리스트 (Circular Doubly Linked List)
@@
본 프로젝트는 더미 노드(Dummy Node)를 활용하여 설계된 **원형 이중 연결 리스트(Circular Doubly Linked List)** 라이브러리입니다. 기존 선형 리스트나 변칙적인 연결 리스트 구조에서 발생할 수 있는 경계면 조건 및 예외 처리를 최적화하고, 데이터 노드 간의 끊임없는 양방향 순환이 가능하도록 구현되었습니다.

---

## 1. 주요 특징 및 구조적 장점

* **더미 노드(Dummy Node) 기반 설계**
  * 리스트 초기화 시 데이터가 없는 더미 노드를 생성하여 기준점으로 삼습니다.
  * 노드의 삽입 및 삭제 시 '리스트가 비어있는 경우(Empty)'나 '맨 앞/맨 뒤에 추가하는 경우'에 발생할 수 있는 Null Pointer 예외 및 까다로운 조건 분기를 최소화합니다.
* **완전한 원형 양방향 순환**
  * 마지막 데이터 노드의 `next`는 더미 노드를 거쳐 첫 번째 데이터 노드로 자연스럽게 연결되며, 첫 번째 데이터 노드의 `prev` 역시 역방향으로 완벽하게 제어됩니다.
* **안전한 경계면 제어 및 포인터 보호**
  * 주목 포인터(`crnt`)가 리스트를 순회할 때 사용자가 더미 노드의 메모리 영역을 직접 밟지 않도록 자동으로 건너뛰는(Jump) 예외 처리 로직이 포함되어 있습니다.
  * 외부 사용자가 의도치 않게 더미 노드 자체를 해제(`free`)하여 전체 데이터 링크가 파괴되는 상황을 내장된 방어 코드로 완벽하게 차단합니다.

---

## 2. 파일 구성

* `member.h` / `member.c`: 리스트에 저장될 사용자 정의 데이터 타입(`member` 구조체) 및 출력 함수 정의
* `CircDbLinkedList.h`: 원형 이중 연결 리스트 구조체(`Dnode`, `Dlist`) 선언 및 외부 공개 API 명세 (Doxygen 스타일 적용)
* `CircDbLinkedList.c`: 라이브러리 핵심 기능(삽입, 삭제, 순환 이동, 검색, 자원 해제 및 시각적 디버깅 출력) 구현
* `main.c`: 함수별 정상 동작 및 예외 처리 방어 코드를 철저하게 검증하기 위한 명세서 형식의 테스트 시나리오 프로그램

---

## 3. 핵심 API 기능 요약

### 초기화 및 해제
* `init_Dlist`: 리스트를 초기화하고 더미 노드 고리를 생성합니다.
* `clear`: 더미 노드를 제외한 모든 진짜 데이터 노드를 안전하게 해제합니다.
* `terminate_Dlist`: 내부 데이터를 모두 비운 후 기준점인 더미 노드까지 완벽하게 해제하여 메모리 누수를 방지합니다.

### 데이터 삽입 및 삭제
* `insert_after`: 지정된 임의의 노드 바로 뒤에 양방향 링크를 유지하며 새 데이터를 끼워 넣습니다.
* `insert_front` / `insert_rear`: `insert_after`를 재사용하여 각각 리스트의 맨 앞과 맨 뒤에 데이터를 효율적으로 배치합니다.
* `remove_Dnode`: 대상 노드의 앞뒤 링크를 재연결하여 분리한 후 안전하게 해제합니다. (더미 노드 삭제 시도 시 차단 및 경고 문구 출력)
* `remove_front` / `remove_rear` / `remove_current`: 리스트가 비어있지 않을 때 각각 맨 앞, 맨 뒤, 그리고 현재 주목 중인 노드를 안전하게 제거합니다.

### 탐색 및 주목 이동
* `search`: 주어진 비교 함수 포인터를 활용하여 조건에 일치하는 데이터를 순방향으로 검색하고 `crnt`를 갱신합니다.
* `move_next` / `move_prev`: 주목 포인터(`crnt`)를 순방향/역방향으로 이동시키되, 경계면(더미 노드) 도달 시 자연스럽게 반대편 데이터 영역으로 점프시켜 원형 순환을 완성합니다.
